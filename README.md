# jim.software

Personal landing page and portfolio.

**Live at [jim.software](https://jim.software)**

## Sections

- **CorridorCanvas** — Three.js procedural corridor background
- **Hero** — name + role
- **BioCard** — career summary
- **Portfolio** — project showcase
- **Signal** — daily-logger feed
- **Teaser** — featured work
- **Footer** — contact + links

## Tech

Vite + React + TypeScript + Three.js. Deployed on Vercel.

Part of the [ojfbot](https://github.com/ojfbot) org.
